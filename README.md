# Everyday Algorithms 🌍💻

## 📖 About This Project
The world is full of invisible algorithms. Instead of relying solely on standard competitive programming platforms, I decided to start looking around me. 

This repository is a daily diary of problem-solving. Every day, I take a real-life scenario I encounter, extract the mathematical and logical constraints, formulate a strict problem statement, and code a solution.

## 🧠 The Daily Process
1. **Observe:** Find a quirk, inefficiency, or interesting pattern in daily life.
2. **Formulate:** Define the inputs, rules, and constraints to turn the observation into a concrete problem.
3. **Solve:** Write clean, efficient code to crack the logic.
